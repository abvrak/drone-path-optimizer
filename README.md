# drone-path-optimizer
Automatyzacja wyznaczania optymalnych tras 3D dla dronów w Lublinie przy użyciu ArcPy. System integruje dane o barierach architektonicznych (BDOT10k), rzeźbie terenu (NMT) oraz dynamiczne parametry pogodowe z OpenWeatherMap API do analizy kosztów przelotu.
